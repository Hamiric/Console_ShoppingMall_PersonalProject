# Console ShoppingMall
개인 과제 - 콘솔 쇼핑몰

## 프로젝트 소개
아래의 기능들을 가진 콘솔 쇼핑몰 프로그램

<필수 기능> 
1. 상품 목록
2. 장바구니 담기
3. 장바구니 총 금액 나타내기
4. 프로그램 종료

<도전기능>
1. 프로그램 종료 시 묻기
2. 장바구니 초기화
3. 장바구니 상품의 목록과 가격 나타내기
4. 자유 구현
